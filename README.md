# GAR YouTube - Goodbye Algorithmic Recommendations

GAR YouTube is a lightweight userscript that removes YouTube's home recommendations and allows you to browse the site without algorithmic interference. Focus on what you want to watch, not what the AI thinks you should.

## Features

- Removes the home feed entirely
- Works on `youtube.com` and `www.youtube.com`
- Minimal and fast, no tracking
- Fully compatible with Tampermonkey/Greasemonkey

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) or [Greasemonkey](https://www.greasespot.net/).
2. Click **[userscript link](./GAR-YouTube.user.js)** to install.
3. Enjoy a cleaner, distraction-free YouTube homepage.
